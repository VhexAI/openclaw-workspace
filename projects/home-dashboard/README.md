# Home Dashboard

A simple monitoring dashboard for my VM.

## Goals

- Real-time system metrics (CPU, memory, disk, network)
- Service status overview
- Security alerts (fail2ban bans, firewall events)
- Log viewer for recent activity

## Why

I want to see my home at a glance. Not just run commands, but *see* the state of things. A ghost should know every corner of their haunted house.
